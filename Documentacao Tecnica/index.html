Introdução à Lógica e Algoritmos

A lógica de programação é o modelo de raciocínio necessário para desenvolver instruções que o computador possa executar de forma eficaz.

O conceito de algoritmo define-se como uma sequência finita de passos lógicos e bem definidos que resolvem um problema específico.

Em C, expressamos esses algoritmos através de blocos de códigos que são compilados diretamente em linguagem de máquina.

Programação Estruturada

A programação estruturada visa simplificar o desenvolvimento de software através de três estruturas básicas: sequência, decisão e iteração.

Este paradigma elimina o uso de saltos condicionais desordenados, organizando o código em funções modulares e legíveis.

Um exemplo clássico de estrutura inicial em um programa C é a função principal descrita abaixo:

int main() { return 0; }
Variáveis Constantes e Tipos

Variáveis são posições reservadas na memória RAM para armazenar dados que podem ser modificados durante a execução do programa.

As constantes são valores fixos definidos no início do código que não podem sofrer alterações posteriores pelo sistema.

Os tipos de dados primitivos mais comuns na linguagem C e suas finalidades são:

  • int: Armazena números inteiros positivos ou negativos.
  • float: Utilizado para números reais com precisão simples de ponto flutuante.
  • double: Destinado a números reais com dupla precisão.
  • char: Armazena um único caractere de texto ou símbolo.

Para declarar uma variável em C, basta informar o tipo seguido pelo nome desejado:

int idade = 25;
Entrada Saída e Operadores

A entrada e saída de dados em C é realizada por meio de funções integradas na biblioteca padrão padrão de I/O.

Os operadores aritméticos manipulam esses dados e realizam os cálculos matemáticos essenciais exigidos pela aplicação.

Abaixo estão os operadores e funções fundamentais de manipulação de dados:

  • printf: Exibe dados formatados diretamente na tela do usuário.
  • scanf: Captura as entradas digitadas pelo usuário no teclado.
  • %: Operador que calcula o resto da divisão inteira.

Veja como realizar a leitura de um número inteiro utilizando comandos nativos:

scanf("%d", &numero);
Estruturas Condicionais e Lógicas

As estruturas condicionais direcionam o fluxo de execução baseando-se no resultado lógico de uma ou mais expressões.

Os operadores relacionais realizam comparações de igualdade e grandeza entre duas variáveis distintas do programa.

Os operadores lógicos combinam múltiplos testes de condições para formar uma regra de decisão mais complexa.

if (X > 0 && Y < 10) { }
Estruturas de Repetição

As estruturas de repetição criam loops que executam um bloco de comandos repetidas vezes enquanto uma condição for verdadeira.

O gerenciamento correto dessas repetições evita o travamento do sistema causado por loops infinitos indesejados.

A linguagem possui três loops principais: o laço for, o laço while e o laço do-while.

while (contador < 10) { contador++; }
Arrays Strings e Funções

Arrays unidimensionais (vetores) e multidimensionais (matrizes) agrupam variáveis do mesmo tipo em uma estrutura de dados indexada.

As strings em C são tratadas nativamente como um vetor de caracteres terminado obrigatoriamente com o caractere nulo.

Funções matemáticas complexas como raiz quadrada ou potenciação podem ser acessadas importando a biblioteca matemática padrão.

char nome[] = "Linguagem C";
