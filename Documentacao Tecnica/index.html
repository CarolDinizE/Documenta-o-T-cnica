Introdução à Lógica e Algoritmos

A lógica de programação ajuda a pensar na ordem correta para resolver um problema.

Um algoritmo é simplesmente uma receita com o passo a passo para o computador executar.

Programação Estruturada

A programação estruturada organiza o código em uma sequência lógica e limpa.

Evitamos códigos bagunçados dividindo nosso programa em blocos menores chamados funções.

int main() { return 0; }
Variáveis e Tipos de Dados

Variáveis servem para guardar informações na memória do computador.

As constantes guardam valores que nunca mudam durante o programa.

Os principais tipos de dados em C são:

  • int: guarda números inteiros.
  • float: guarda números com vírgula.
  • char: guarda uma letra ou caractere.
int idade = 18;
Entrada, Saída e Operadores

Usamos comandos para mostrar mensagens na tela e ler o que o usuário digita.

Os operadores aritméticos servem para fazer contas matemáticas básicas.

  • printf: mostra um texto na tela.
  • scanf: lê um dado do teclado.
printf("Olá Mundo");
Estruturas Condicionais

As estruturas condicionais ajudam o programa a tomar decisões usando o "if" (se).

Usamos operadores relacionais e lógicos para comparar valores.

if (nota >= 6) { }
Estruturas de Repetição

As estruturas de repetição servem para rodar o mesmo código várias vezes.

Isso evita que a gente precise copiar e colar o mesmo comando muitas vezes.

while (i < 10) { }
Arrays, Strings e Funções

Arrays e vetores guardam vários dados juntos do mesmo tipo.

Uma string em C é apenas uma lista de letras (caracteres) juntas.

char nome[20];
